# 电商产品出图流程

状态：进行中，已加入第一轮原始素材提取规范和家居 Bento 卖点海报分支

## 1. 流程定位

本流程用于把 Eagle 中的产品素材，整理成可执行的电商出图任务。适用范围包括主图、卖点图、材质图、结构图和详情页素材。

核心原则：

- 先理解产品，再写提示词。
- 先判断原图用途，再决定是否生图。
- 原图只能做参考和证据，不能直接当作电商成品图。
- 先提取，再优化，再排版；不从原图一步跳到电商图。
- 出图调性、色调和排版必须由目标人群、价格档位和主卖点决定。
- 生图只负责生成干净视觉素材，文字卖点和版式尽量后期排版。
- 不直接让模型生成复杂中文详情页，避免错字、假卖点和版式失控。
- 每次出图前必须有产品信息提取表、素材用途表和生图任务表。
- 正式商品图只走自有日本服务器 GPT Plus 的 image-2 通道；不使用 free 通道。
- 第一轮必须先做原始素材提取，不直接做成品海报。
- 第一轮提示词必须按“目的 / 参考图 / 提示词”输出，提示词块能直接复制进生图提示框。
- Eagle 文件夹可能混入其他品类素材，先筛出当前产品相关图和视频，不把无关素材混进参考图。
- 有视频时先判断视频能提供什么信息，再决定抽帧或跑视频分析；涉及外部模型调用前先确认。
- 产品本体上的真实 logo、吊牌、侧标、产品标签属于产品识别信息，默认保留；背景品牌墙、水印、二维码、价格牌和额外宣传文字才默认去掉。
- 第一轮主图不是最终白底成品图，而是方便后期放进环境、场景或版式里的干净产品主体素材；不要默认生成死板白底孤立图。
- 家居产品进入卖点海报阶段时，优先参考 `docs\家居产品Bento卖点海报工作流.md`；学习 Bento 工作流的拆分方法，不照搬 3C/电器参数表达。

## 2. 标准工作顺序

1. 产品信息提取
2. 目标人群和价格档位判断
3. 卖点优先级排序
4. 电商视觉调性确定
5. 现有素材盘点
6. 原图用途分配
7. 筛出无关素材，识别图片和视频信息
8. 第一轮原始素材提取提示词
9. 原图预处理和素材提取
10. 图片类型规划
11. 判断进入主图 / 材质图 / 结构图 / 家居 Bento 卖点海报哪条分支
12. 提示词编写
13. 和卡卡确认产品卖点、人群、视觉调性和缺失信息
14. 确认出图通道、比例、分辨率和参考图
15. 小批量试生成
16. 结果筛选和复盘
17. 二轮精修、单模块替换或后期排版
18. 确认后再导入 Eagle 或建立产品文件夹

## 3. 产品信息提取表

每个产品先写清楚：

| 字段 | 内容 |
|---|---|
| 产品名称 | 准确 SKU 或临时名称 |
| 品类 | 例如床垫、沙发、椅子、灯具 |
| 外观特征 | 颜色、形状、厚度、纹理、边线、结构 |
| 核心材质 | 面料、木材、金属、黄麻、弹簧等 |
| 结构层次 | 可见结构和需要解释的隐藏结构 |
| 使用场景 | 主卧、儿童房、酒店、公寓等 |
| 目标人群 | 老人、年轻人、家庭、租房、酒店、儿童等 |
| 价格档位 | 低价实惠、高性价比、中高端、高端 |
| 核心卖点 | 可从产品本身推出的卖点 |
| 风险卖点 | 需要报告或证据支撑的卖点 |
| 禁止误读 | 模型容易画错的地方 |

## 4. 人群、调性和排版策略

电商图不是只看产品，还要看卖给谁。确定人群后，先定视觉方向，再定出图任务。

| 人群/定位 | 色调 | 画面气质 | 排版方式 | 文案气质 |
|---|---|---|---|---|
| 老人护脊 | 米白、米棕、浅木色、暖灰 | 温暖、自然、稳定、家用感 | 大字、少信息块、强重点、生活场景和材料说明优先 | 朴实、安心、实惠、不夸张 |
| 年轻软睡感 | 浅奶油、浅灰、柔和低饱和 | 柔软、舒适、生活方式 | 留白多、氛围图多 | 轻松、治愈、舒适 |
| 高端酒店感 | 深灰、黑金、冷白、低饱和 | 高级、克制、精致 | 大图压场、细节少而准 | 品质、工艺、静音、支撑 |
| 低价实用 | 白底、蓝色/红色强调、清晰对比 | 直给、清楚、便宜有料 | 标题明确、参数明显、卖点图标化 | 划算、耐用、真材实料 |
| 儿童青少年成长睡眠 | 白色、浅米色、浅木色、淡蓝、淡绿、暖灰 | 清爽、明亮、健康、安心 | 主图干净，结构和材质图先无文字，后期再加清晰卖点 | 软硬适中、透气、舒适承托、成长睡眠 |

排版先分层：

1. 第一眼：产品是什么，适合谁。
2. 第二眼：为什么适合这个人群。
3. 第三眼：材料和结构证明。
4. 第四眼：检测报告、参数、售后等信任背书。

## 5. 现有素材用途分配

素材先分类，不直接混用：

| 素材类型 | 主要作用 | 适合生成什么 | 不适合做什么 |
|---|---|---|---|
| 全景产品图 | 锁定整体外观、比例、角度 | 白底主图、详情页首屏素材、场景图主体 | 直接生成材质细节 |
| 局部细节图 | 锁定面料、包边、拉链、缝线 | 面料细节图、工艺卖点图 | 直接代表完整产品 |
| 开窗/剖面图 | 表现内部结构或隐藏材质 | 卖点图、结构说明图 | 生成全景主图 |
| 材质图 | 锁定材料颜色和纹理 | 材质微距图、材质小样 | 直接当成成品图 |
| 旧详情页 | 提取文案层级和模块结构 | 详情页策划参考 | 直接图生图生成新详情页 |
| 视频 | 补充动态展示、手感、厚度、侧围、使用动作 | 抽关键帧、提取产品事实、补充提示词约束 | 未确认前直接整段送外部模型或当成唯一参考 |

## 6. 分步出图链路

所有产品图都按三段式推进：

### 6.1 第一轮原始素材提取阶段

目标：从原图里还原可用基础素材，不做成品海报，不做人群营销，不直接排版。

输出：

- 产品主图还原提示词
- 产品结构图还原提示词
- 产品材质图还原提示词
- 材质和纹理清单
- 可见外观特征
- 原图问题清单
- 可用卖点和待确认卖点

第一轮必须先还原三类素材：

| 类型 | 目的 | 参考图选择 | 默认要求 |
|---|---|---|---|
| 产品主图 | 还原产品整体外观，作为后期排版或置入环境的主体 | 全景产品图为主，局部图只辅助锁定纹理 | 干净自然背景或浅环境、真实比例、保留产品本体 logo/标签、无人物、无多余道具 |
| 产品结构图 | 还原内部层次或开窗结构，作为后期标注素材 | 开窗/剖面图为主，全景图锁定外观，材质图锁定关键材料 | 无文字、无箭头、无编号、结构可信 |
| 材质图 | 还原材料颜色、纹理、质感和层次 | 材质图或局部裁剪图为主 | 干净近景或材质小样，无勾选符号、无宣传文字 |

给卡卡的第一轮输出格式固定为：

```text
目的：
参考图：
提示词：
```

其中“提示词”必须是一整段可以直接复制进生图提示框的内容；“目的”和“参考图”只用于说明，不放进生图框也能独立理解。

### 6.2 优化阶段

目标：把原图变成可用于电商排版的干净素材。

常见任务：

- 背景清理
- 背景品牌墙、价格牌、水印、二维码和无关宣传文字去除
- 局部材质裁剪
- 主体角度和光线优化
- 生成白底主图、材质小样、结构素材

注意：

- 这一阶段默认不生成中文文字。
- 这一阶段不做完整详情页。
- 优化阶段只在第一轮素材还原方向正确后进行。

### 6.3 排版阶段

目标：把已优化素材做成真正的电商图。

常见任务：

- 主图构图
- 卖点图标题和短文案
- 材质说明图
- 结构拆解图
- 检测报告背书模块
- 详情页长图节奏

注意：

- 文案和参数以后期排版为主。
- 检测报告类内容必须以真实报告为准。

### 6.4 家居 Bento 卖点海报阶段

使用时机：

- 已经有可用产品主图、材质图、细节图或结构图。
- 需要把单个家居产品做成详情页卖点海报、主图卖点图或小红书种草图。
- 产品卖点已经从功能参数，转化为家居购买决策维度。

执行规则：

- 参考 `docs\家居产品Bento卖点海报工作流.md`。
- 先由家居视觉策划总监 Agent 输出《家居产品高转化视觉执行方案》。
- 再由家居生图 Prompt 工程师 Agent 输出 Hero 和每个 Bento 模块的独立提示词。
- 生图阶段优先生成无字底图，标题、副标题、参数、图标后期排版。
- 每个模块必须独立保存标题、副标题、视觉描述、比例、提示词和质检结果。
- 单个模块不满意时，只重生该模块，不整张重做。

家居 Bento 模块优先从以下维度选择：

| 模块维度 | 说明 |
|---|---|
| 空间适配 | 产品放进真实家居空间后的尺度、留白和搭配关系 |
| 材质质感 | 木纹、布艺、皮革、黄麻、金属、石材等微距质感 |
| 工艺细节 | 缝线、包边、边角、五金、支脚、收口 |
| 尺寸比例 | 高度、厚度、坐深、离地高度、通道空间 |
| 舒适体验 | 坐感、睡感、支撑、透气、回弹、包裹 |
| 生活方式 | 亲子、阅读、围坐、会客、安静入眠等使用场景 |
| 清洁维护 | 防污、易擦、耐磨、宠物友好、可拆洗 |
| 风格搭配 | 奶油风、原木风、轻法式、侘寂风等空间适配 |
| 信任背书 | 环保、检测、承重、材料、安装和售后 |

## 7. 图片类型规划

### 7.1 全景主图

使用时机：

- 需要商品首图、列表图、详情页首屏。
- 原图已有完整产品，只是背景杂、光线差、文字干扰。

参考图选择：

- 优先使用完整产品全景图。
- 最多补一张局部图锁定纹理，不要混入旧详情页。

生成目标：

- 产品完整、角度稳定、白底或浅灰棚拍。
- 无文字、无 logo、无人物、无多余道具。
- 保留真实比例、侧面结构、边线和面料纹理。

### 7.2 细节图

使用时机：

- 需要表现面料、包边、拉链、缝线、侧围等可见工艺。

参考图选择：

- 使用对应局部特写。
- 不加入全套复杂素材，避免模型重构错误。

生成目标：

- 近景真实、纹理清晰、光线干净。
- 不承担完整产品说明。

### 7.3 材质图

使用时机：

- 需要解释天然黄麻、乳胶、海绵、木材、金属等材料本身。

参考图选择：

- 使用材质原图或从原图裁出来的局部图。

生成目标：

- 干净材料样本或微距照片。
- 不出现宣传文字、勾选符号、假检测元素。

### 7.4 结构图

使用时机：

- 需要说明床垫层次、弹簧结构、支撑层等隐藏信息。

参考图选择：

- 全景图锁定外观。
- 开窗图锁定真实结构。
- 材质图锁定关键材料。

生成目标：

- 无文字、无箭头、无编号的干净剖面素材。
- 文案和标注后期再叠加。

### 7.5 第一轮素材还原图

使用时机：

- 刚拿到一个产品文件夹，还没有确认主图、结构图和材质图可用。
- 原图有文字、品牌墙、旧详情页、环境杂乱或画质不统一。

参考图选择：

- 主图还原：只用全景产品图，最多加一张局部纹理图。
- 结构还原：开窗/剖面图为主，全景图辅助锁定外观，材质图辅助锁定材料。
- 材质还原：材质原图或无损裁剪局部为主，不混入完整详情页。

生成目标：

- 先得到干净基础素材，不做卖点海报，不把主图做成最终白底成品。
- 产品本体上的真实 logo、侧标、吊牌、产品标签要保留；背景 logo、水印、价格、勾选符号和额外宣传字要去掉。
- 不出现人物，除非卡卡明确要求。
- 不让模型自行改产品款式、结构、比例、纹理和颜色。
- 主图背景优先使用干净浅环境、轻场景、透明感棚拍或可后期抠图的自然灰底，不默认要求纯白死背景。

### 7.6 详情页卖点图

使用时机：

- 已经有可用主图、细节图、材质图和结构图后。

生成目标：

- 生图阶段只生成背景和主体视觉。
- 详情页的标题、卖点、参数、图标由后期设计完成。

## 8. 原图预处理规则

生图前优先处理：

- 去掉原图中文字、价格牌、水印、勾选符号。
- 但产品本体上的品牌侧标、吊牌、织唛和真实产品 logo 不属于水印，默认保留。
- 裁出关键局部材质，减少无关背景。
- 按用途拆分参考图，不把所有图一起喂给模型。
- 对重要材质单独裁剪一张参考图。

不要做：

- 不把原图直接套成电商成品图。
- 不把旧详情页直接作为强参考。
- 不让模型生成大量中文。
- 不把全景、材质、结构、版式混成一个 prompt。

### 8.1 Eagle 文件夹和视频处理规则

打开 Eagle 文件夹后先做素材盘点：

- 统计当前文件夹里的图片、视频和其他文件数量。
- 先筛选当前产品相关素材，忽略混入的其他品类。
- 识别哪些是原始参考，哪些是 AI 生成候选，哪些是旧详情页。
- 视频只作为补充事实来源，不默认替代原图。

有视频时按优先级处理：

1. 先看视频缩略图、文件名、时长、比例和已有 annotation。
2. 如果本机有 ffmpeg 或 OpenCV，可先抽 5 到 12 张关键帧。
3. 如果需要更完整的动态信息，再用 Eagle video-analyzer 做视频语义分析。
4. 涉及外部模型调用、可能产生费用或上传视频时，先向卡卡确认。

视频重点提取：

- 产品整体外观、厚度、侧围、包边。
- 面料纹理、开窗结构、材质层次。
- 手按、翻折、回弹、透气、支撑等动态展示。
- 和静态图片冲突的地方，优先标记为待确认，不直接写死进提示词。

## 9. 提示词结构

每个生图任务使用固定结构：

```text
任务类型：
阶段：提取 / 优化 / 排版
参考图角色：
目标人群：
价格定位：
视觉调性：
主体描述：
必须保留：
允许优化：
构图与镜头：
背景与光线：
材质要求：
输出用途：
禁止项：
```

提示词必须明确：

- 这张图是主图、细节图、材质图还是结构图。
- 这张图处在提取、优化还是排版阶段。
- 每张参考图分别负责什么。
- 哪些元素必须保留。
- 哪些元素必须删除。
- 是否允许出现文字，默认不允许。

第一轮原始素材提取时，给卡卡的展示格式必须改为：

```text
目的：
参考图：
提示词：
```

“提示词”部分要求：

- 能直接复制进生图提示框。
- 不包含解释语气、教程语气和内部分析。
- 明确这是“素材还原 / 提取”，不是成品海报。
- 明确参考图各自负责什么。
- 明确保留产品外观、结构、材质、比例和纹理。
- 明确禁止背景文字、水印、价格、勾选符号和无关人物。
- 明确保留产品本体真实 logo、侧标、吊牌、织唛和产品识别标签。

## 10. 出图通道规则

正式商品图任务必须在生图任务表里写清楚：

| 字段 | 规则 |
|---|---|
| provider | 自有日本服务器 GPT Plus image-2 通道 |
| model | image-2 / gpt-image-2 / codex-gpt-image-2，按当前可用配置确认 |
| 禁用通道 | free、本地未启动端口、临时测试供应商 |
| 出图前检查 | 确认 active provider 或请求参数没有误走 free |
| 分辨率 | 按主图、卖点图、详情横幅分别确认，不默认乱用 |

若本地系统默认 provider 不是日本 Plus，必须在任务请求中显式指定日本 Plus/OAuth GPT image-2 通道。

## 11. 小批量试生成规则

每轮最多先跑 3 到 6 张：

| 任务 | 数量 | 目的 |
|---|---:|---|
| 主图 | 1-2 | 看产品比例和背景清洁度 |
| 细节图 | 1-2 | 看材质和局部真实度 |
| 材质图 | 1 | 看材料是否被误画 |
| 结构图 | 1 | 看层次是否可信 |

筛选标准：

- 产品像不像原产品。
- 是否多出不该有的东西。
- 材质是否正确。
- 是否有文字污染。
- 是否适合后期叠文案。

## 13. 黄麻床垫当前应用规则

黄麻床垫应先形成以下三张表，再继续出图：

1. 产品信息提取表
2. 现有素材用途表
3. 人群/调性/排版策略表
4. 生图任务表

当前定位：

| 字段 | 结论 |
|---|---|
| 目标人群 | 孩子、青少年、成长阶段家庭用户 |
| 价格定位 | 实惠、高性价比 |
| 主卖点 | 软硬适中、适合孩子和青少年 |
| 次卖点 | 透气性好、黄麻支撑、独立袋装弹簧、舒适填充 |
| 信任背书 | 检测报告和材料说明需以真实资料为准 |
| 色调建议 | 白色、浅米色、浅木色、淡蓝、淡绿、暖灰，整体清爽明亮 |
| 排版建议 | 先做无文字主图、结构图、材质图，再后期加“软硬适中、透气、成长睡眠”卖点 |
| 出图方式 | 原图先做参考和素材提取，再优化成干净素材，最后后期排版 |

建议第一轮任务：

| 编号 | 图型 | 参考图 | 目标 |
|---|---|---|---|
| A1 | 产品主图还原 | 全景产品图 | 还原床垫整体外观，清理背景，禁止文字和人物 |
| A2 | 产品结构图还原 | 开窗图 + 全景图 + 黄麻材质图 | 还原面层、黄麻层、透气层、弹簧层，禁止文字箭头编号 |
| A3 | 材质图还原 | 黄麻材质图 + 面料局部图 + 开窗图 | 提取黄麻、白色针织面料、透气层材质小样 |
| A4 | 视频信息提取 | 50 秒竖屏视频 | 补充面料纹理、厚度、侧围、手感和动态展示信息 |

第一轮输出必须按“目的 / 参考图 / 提示词”写，提示词可直接进生图框。
