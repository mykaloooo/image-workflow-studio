# awesome-gpt-image-2 家具电商候选模板包

来源：<https://github.com/freestylefly/awesome-gpt-image-2>

整理时间：2026-05-14 18:43

用途：给 `image-workflow-studio` 做家具电商出图模板候选，不直接覆盖现有 `backend/prompt_templates.json`。

## 总判断

这个仓库适合吸收为“提示词结构库”，不适合整体替换我们的系统。

本次只筛选并改写与家具电商强相关的方向：

- 家具电商主图
- 深色木自然光通透主图
- 浅色产品明暗层次主图
- 多角度系列图
- 材质细节特写
- 家具场景图
- 卖点信息图
- 结构拆解图
- 材质情绪板
- 品牌 Campaign 图
- 参考图反推结构化 prompt

## 必须优先保留的卡卡规则

- 沙发类：不要白底，暖色纯净背景 + 景深棚拍效果。
- 浅色产品：背景要比主体略深或更暖，必须拉开明暗层次。
- 黑胡桃木等深色木：不要太暖，不要发闷、发灰、发黑，要自然光通透。
- 多角度：正面、45 度、侧面至少要有。
- 主体保真：产品结构、比例、材质、颜色、部件数量必须保留。
- 生成完成后：正式批量图应导入 Eagle 对应产品文件夹。

## 候选模板清单

### 1. 沙发电商主图 · 暖色景深棚拍

适合：真皮沙发、功能沙发、布艺沙发、休闲椅。

核心：保留沙发结构，主体约 50%，暖色纯净棚拍背景，有景深，不要白底。

```text
电商主图，严格保留参考图中沙发的整体结构、形状、比例、材质、颜色、扶手、靠背、坐垫、缝线、金属件或木脚细节，不添加、不删除、不重塑任何部件。

主体为[产品名称/材质颜色]，视角为[正面/45度/侧面]，主体占画面约50%，构图居中稳定，适合电商列表页和产品目录。背景为暖色纯净棚拍背景，有柔和景深和自然阴影，不要白底，不要灰脏背景，不要复杂室内杂物。

光线为柔和商业摄影灯光，前侧主光 + 轻微轮廓光，真实呈现皮革/布艺/木质/金属质感，画面干净高级，高清细节，商业摄影级质感。

约束：产品必须与参考图一致；不要改变沙发款式；不要改变坐垫数量；不要生成文字、Logo、水印、人物、手、装饰抱枕以外的随机物；不要夸张反光；不要过度广角变形。
```

### 2. 黑胡桃木家具主图 · 自然光通透

适合：黑胡桃木茶桌、餐桌、椅子、博古架、柜类。

核心：明亮自然光，浅暖灰/自然光色背景，通透不发闷。

```text
电商产品主图，严格保留参考图中[产品名称]的结构、比例、轮廓、腿型、边缘厚度、木纹方向、拼接细节和真实木质质感。产品材质为[黑胡桃木/深色木]，表面有自然木纹、细腻哑光或半哑光光泽。

画面采用明亮自然光棚拍风格，像阳光透过大窗户照进摄影棚，背景为高明度浅暖灰或自然光色，通透、干净、有空气感。不要太暖，不要发闷，不要发灰，不要发黑，不要白底。

视角为[正面/45度/侧面/俯视]，主体占画面约50%，构图稳定，地面有柔和自然阴影，突出深色木与明亮背景的层次。

约束：不得改变产品造型、尺寸比例、木纹方向和部件数量；不得添加人物、文字、Logo、水印、杂乱软装；不要让背景抢主体；不要把深色木变成黑块。
```

### 3. 浅色家具主图 · 明暗层次对比

适合：米白沙发、奶油色椅、浅木家具。

核心：背景不能和主体同色，必须拉开明暗层次。

```text
电商主图，严格保留参考图中[浅色产品名称]的结构、比例、轮廓、材质和颜色，不改变产品款式。主体为[米白/奶油白/浅灰/浅木色]，需要清晰呈现边缘线、缝线、织物纹理、皮革纹理或木纹细节。

背景不能与主体同色，背景与前景主体必须拉开明暗层次和色温层次。使用略深的暖灰、浅咖、柔和米棕或低饱和暖色棚拍背景，带浅景深和柔和地面阴影，让浅色主体轮廓清楚。不要白底，不要纯浅米白背景，不要让产品融进背景。

主体占画面约50%，视角为[正面/45度/侧面]，构图干净，高级电商棚拍质感。

约束：不要生成文字、Logo、水印、人物、杂乱道具；不要改变产品颜色为深色；不要过曝；不要丢失浅色材质纹理。
```

### 4. 家具多角度系列 · 正面45度侧面

适合：同一产品批量出正面、45 度、侧面。

```text
为参考图中的同一款[家具产品]生成[正面/45度/侧面]电商主图。必须保持同一产品身份：结构、比例、材质、颜色、坐垫数量、扶手形状、靠背高度、脚架样式、缝线和边缘细节都与参考图一致。

本张只输出一个角度：[正面/45度/侧面]。不要生成三视图合集，不要生成多件不同产品。主体占画面约50%，构图居中，背景和光线与同系列其他角度保持一致。

背景策略：[沙发用暖色纯净景深棚拍 / 深色木用自然光通透浅暖灰 / 浅色产品用略深暖灰拉开层次]。

高清商业摄影，真实材质，柔和阴影，电商目录级质感。

约束：不要改变产品造型；不要添加文字、Logo、水印、人物和杂乱道具；不要让每个角度看起来像不同型号。
```

### 5. 家具材质细节 · 局部特写

```text
根据参考图生成[家具产品]的材质局部特写图，聚焦[扶手/坐垫/靠背/缝线/木纹/边角/五金/脚架]区域。必须保留参考图中的真实材质、颜色、纹理方向、缝线或连接结构，不改变产品设计。

构图干净，可用于电商详情页材质说明。不要生成文字标注，不要添加手、工具、尺子、Logo、水印。

约束：局部必须来自同一产品；不要把纹理做得过度夸张；不要让材质变色；不要生成无法识别产品关系的抽象背景。
```

### 11. 参考图反推 · 家具Prompt结构化

适合：把外部好图沉淀成我们自己的提示词金样板。

```text

请按以下字段输出：
1. 产品身份：品类、风格、核心造型、比例特征。
2. 必须保留：结构、部件数量、材质、颜色、纹理、工艺、边缘和脚架等关键点。

5. 材质表达：皮革、布艺、木材、金属、石材等如何呈现。
6. 输出用途：电商主图、详情图、场景图、卖点图。
7. 避免项：不要改变产品结构，不要白底，不要人物，不要Logo水印，不要乱码文字，不要复杂杂物。

最后给出一段可直接复制使用的最终中文 prompt。最终 prompt 必须优先遵守卡卡家具出图规则：沙发类暖色背景+景深棚拍；浅色产品背景要拉开明暗层次；黑胡桃木等深色木要自然光通透、不发闷。
```

## 接入建议

先不要自动合并到正式模板库。建议先拿 2-3 个产品做小批测试：

1. 一个沙发：测“暖色景深棚拍”和“三角度系列”。
2. 一个黑胡桃木茶桌：测“自然光通透”。
3. 一个浅色沙发或浅木产品：测“明暗层次对比”。

测试效果稳定后，再把候选 JSON 合并进 `backend/prompt_templates.json`。
