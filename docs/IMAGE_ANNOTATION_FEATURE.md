# 图片标注编辑功能

## 功能概述

新增的图片标注编辑功能允许你直接在生成的图片上进行标注和编辑，包括：
- ✏️ 画笔工具 - 自由绘制
- ↗️ 箭头工具 - 添加指向性标注
- ▭ 矩形工具 - 框选区域
- Tt 文字工具 - 添加文字说明
- 撤销/重做 - 支持操作历史
- 删除 - 删除选中的标注对象

## 使用方法

### 1. 打开标注编辑器

1. 在画布上点击任意已生成的图片节点
2. 在弹出的详情面板中，点击 **"✏️ 标注"** 按钮
3. 标注编辑器将以全屏模式打开

### 2. 使用绘图工具

#### 选择工具 (↖️)
- 点击并拖动标注对象来移动
- 点击对象后可以调整大小
- 按 Delete 键删除选中对象

#### 画笔工具 (✏️)
- 按住鼠标左键拖动进行自由绘制
- 可调整线条粗细和颜色

#### 箭头工具 (↗️)
- 点击起点，拖动到终点，释放鼠标
- 自动生成带箭头的指向线

#### 矩形工具 (▭)
- 点击起点，拖动到对角，释放鼠标
- 生成矩形框（透明填充，彩色边框）

#### 文字工具 (Tt)
- 点击画布任意位置
- 输入文字内容
- 双击文字可再次编辑
- 可调整字体大小和颜色

### 3. 调整参数

- **颜色选择器** - 点击色块选择标注颜色
- **线条粗细滑块** - 调整画笔和形状的线条宽度 (1-20px)
- **字体大小滑块** - 调整文字大小 (12-72px)

### 4. 操作历史

- **撤销 (↶)** - 撤销上一步操作
- **重做 (↷)** - 恢复撤销的操作
- **删除 (🗑️)** - 删除当前选中的对象

### 5. 保存标注

1. 完成标注后，点击右上角的 **"保存标注"** 按钮
2. 标注后的图片将自动上传到后端
3. 节点中的图片会更新为标注后的版本
4. 标注后的图片可以继续用于工作流中的其他操作

### 6. 取消编辑

- 点击 **"取消"** 按钮退出编辑器
- 不会保存任何更改

## 技术实现

- **前端库**: fabric.js - 强大的 Canvas 绘图库
- **图片格式**: PNG (无损压缩)
- **保存方式**: 通过 `/api/upload-image` 接口上传到后端
- **存储位置**: `backend/outputs/` 目录

## 注意事项

1. **保留原图**: 标注操作会创建新的图片文件，原图不会被覆盖
2. **文件大小**: 标注后的图片可能比原图略大（因为包含了标注层）
3. **浏览器兼容**: 推荐使用 Chrome 或 Edge 浏览器以获得最佳体验
4. **性能**: 对于超大图片（>4K），标注编辑器可能会有轻微延迟

## 应用场景

- **设计反馈**: 在生成的图片上标注需要修改的地方
- **教学演示**: 添加箭头和文字说明关键部分
- **工作流标记**: 标记图片的特定区域用于后续处理
- **创意迭代**: 在图片上绘制草图指导下一次生成

## 快捷键

- **Delete** - 删除选中对象
- **Ctrl+Z** - 撤销（计划中）
- **Ctrl+Y** - 重做（计划中）
- **Esc** - 取消当前操作（计划中）
