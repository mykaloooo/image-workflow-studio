# 图片工作流工作室 - 使用说明

## 快速开始

### 方式一：Web 开发模式

1. **安装依赖**

```bash
# 安装后端依赖
cd backend
pip install -r requirements.txt

# 安装前端依赖
cd ../frontend
npm install
```

2. **启动服务**

双击 `启动.bat` 文件，会自动启动后端和前端服务。

或者手动启动：

```bash
# 终端1：启动后端
cd backend
python app.py

# 终端2：启动前端
cd frontend
npm run dev
```

3. **打开浏览器**

访问 http://localhost:5173

### 方式二：Electron 桌面应用（推荐）

1. **安装依赖**

```bash
# 安装后端依赖
cd backend
pip install -r requirements.txt

# 安装前端和 Electron 依赖
cd ../frontend
npm install
```

2. **启动 Electron 开发模式**

双击 `启动-Electron开发模式.bat` 文件，会自动启动：
- 后端 Flask 服务
- 前端 Vite 开发服务器
- Electron 桌面应用

或者手动启动：

```bash
# 终端1：启动后端
cd backend
python app.py

# 终端2：启动前端
cd frontend
npm run dev

# 终端3：启动 Electron
cd frontend
npm run electron:dev
```

3. **打包成 EXE**

```bash
cd frontend
npm run electron:build
```

打包后的文件在 `dist` 目录中。

## 使用流程

1. **配置 API**
   - 首次打开会要求输入 Gemini API Key
   - 可选填写代理地址（默认：http://127.0.0.1:10808）

2. **添加节点**
   - 点击工具栏的"➕ 添加节点"按钮
   - 会在画布上创建一个提示词节点

3. **生成图片**
   - 点击节点右侧的 ➕ 按钮
   - 输入提示词
   - 可选上传参考图
   - 选择宽高比和分辨率
   - 点击"生成图片"

4. **继续扩展**
   - 生成的图片会显示为新节点
   - 可以继续点击 ➕ 添加子节点
   - 形成树状工作流

## 功能特性

- ✅ 无限画布（鼠标滚轮缩放，拖拽移动）
- ✅ 树状节点结构
- ✅ 实时生成状态显示
- ✅ 支持多张参考图
- ✅ 自动保存生成的图片到 backend/outputs/

## 快捷键

- **滚轮**：缩放画布
- **拖拽**：移动画布
- **Ctrl + 滚轮**：更快速缩放

## 常见问题

### Q: 生成失败怎么办？
A: 检查：
1. API Key 是否正确
2. 代理是否正常（如果需要）
3. 网络连接是否正常
4. 查看后端终端的错误信息

### Q: 图片保存在哪里？
A: `backend/outputs/` 文件夹

### Q: 如何修改 API 配置？
A: 刷新页面重新配置

## 注意事项

### Electron 打包前的准备

1. **Python 环境**：确保系统已安装 Python，打包后的应用会调用系统的 Python
2. **图标文件**：如需自定义图标，请在 `frontend/public/` 目录添加 `icon.ico` 和 `icon.png`
3. **依赖安装**：首次运行需要安装 Electron 相关依赖：
   ```bash
   cd frontend
   npm install
   ```
