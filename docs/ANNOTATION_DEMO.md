# 图片标注功能演示

## 功能概览

图片标注编辑器是一个全屏的图片编辑工具，集成在无限画布工作流中，允许你直接在生成的图片上进行标注和编辑。

## 界面布局

```
┌─────────────────────────────────────────────────────────────┐
│  工具栏                                                      │
│  [↖️] [✏️] [↗️] [▭] [Tt] │ [🎨] [━━] [Aa] │ [↶] [↷] [🗑️] │ [取消] [保存标注] │

```

## 工具说明

### 左侧工具组 - 绘图工具

| 图标 | 工具名称 | 功能说明 | 使用方法 |
|------|---------|---------|---------|
| ↖️ | 选择工具 | 选择和移动标注对象 | 点击对象进行选择，拖动移动 |
| ✏️ | 画笔工具 | 自由绘制 | 按住鼠标拖动绘制 |
| ↗️ | 箭头工具 | 绘制带箭头的线条 | 点击起点，拖动到终点 |
| ▭ | 矩形工具 | 绘制矩形框 | 点击起点，拖动到对角 |
| Tt | 文字工具 | 添加文字 | 点击位置，输入文字 |

### 中间工具组 - 样式设置

| 图标 | 工具名称 | 功能说明 |
|------|---------|---------|
| 🎨 | 颜色选择器 | 选择标注颜色 |
| ━━ | 线条粗细 | 调整线条宽度 (1-20px) |
| Aa | 字体大小 | 调整文字大小 (12-72px) |

### 右侧工具组 - 操作按钮

| 图标 | 工具名称 | 功能说明 |
|------|---------|---------|
| ↶ | 撤销 | 撤销上一步操作 |
| ↷ | 重做 | 恢复撤销的操作 |
| 🗑️ | 删除 | 删除选中的对象 |

### 最右侧 - 保存/取消

| 按钮 | 功能说明 |
|------|---------|
| 取消 | 退出编辑器，不保存更改 |
| 保存标注 | 保存标注后的图片并更新节点 |

## 使用流程

### 1. 打开标注编辑器

```
画布节点 → 点击图片 → 详情面板 → "✏️ 标注" 按钮 → 全屏编辑器
```

### 2. 进行标注

#### 示例 1: 添加箭头指向
```
1. 点击 "↗️" 箭头工具
2. 在图片上点击起点
3. 拖动到终点
4. 释放鼠标，生成箭头
```

#### 示例 2: 添加文字说明
```
1. 点击 "Tt" 文字工具
2. 点击要添加文字的位置
3. 输入文字内容
4. 点击外部完成编辑
```

#### 示例 3: 框选区域
```
1. 点击 "▭" 矩形工具
2. 点击起点
3. 拖动到对角
4. 释放鼠标，生成矩形框
```

#### 示例 4: 自由绘制
```
1. 点击 "✏️" 画笔工具
2. 选择颜色和线条粗细
3. 按住鼠标拖动绘制
4. 释放鼠标完成
```

### 3. 调整标注

```
1. 点击 "↖️" 选择工具
2. 点击要调整的标注对象
3. 拖动移动位置
4. 拖动角点调整大小
```

### 4. 保存标注

```
1. 完成所有标注
2. 点击 "保存标注" 按钮
3. 等待上传完成
4. 自动关闭编辑器
5. 节点图片更新为标注版本
```

```
设计师生成图片 → 标注需要修改的地方 → 保存 → 继续迭代
```

```
生成示例图片 → 添加箭头和文字说明 → 保存 → 用于教学
```

### 场景 3: 工作流标记
```
生成图片 → 标记特定区域 → 保存 → 作为参考图继续生成
```

### 场景 4: 创意迭代
```
生成初稿 → 在图上绘制修改草图 → 保存 → 用标注图指导下次生成
```

## 技术细节

### 图片处理流程

```
原始图片 → Canvas 渲染 → 添加标注层 → 导出 PNG → 上传后端 → 更新节点
```

### 文件格式

- **输入**: 任何格式的图片 (PNG, JPG, WebP 等)
- **输出**: PNG 格式 (无损压缩)
- **命名**: `annotated_<timestamp>.png`

### 存储位置

```
backend/outputs/annotated_<timestamp>.png
```

### API 调用

```javascript
POST /api/upload-image
Content-Type: multipart/form-data

FormData:
  - image: Blob (标注后的图片)
  - filename: annotated_<timestamp>.png

Response:
{
  "success": true,
  "url": "/api/images/annotated_<timestamp>.png"
}
```

## 性能优化

### 大图片处理

- 自动缩放图片以适应画布
- 保持原始宽高比

- 历史记录限制: 50 步

- 使用 JSON 序列化存储状态

### 渲染优化

- 使用 fabric.js 的硬件加速
- 对象缓存机制
- 按需重绘

## 常见问题

### Q: 标注会覆盖原图吗？
A: 不会。标注后会生成新的图片文件，原图保持不变。

### Q: 可以编辑已保存的标注吗？
A: 标注保存后会合并到图片中，无法再次编辑。如需修改，需要重新标注。

### Q: 支持哪些图片格式？
A: 输入支持所有常见格式，输出为 PNG 格式。

### Q: 标注后的图片可以用于工作流吗？
A: 可以。标注后的图片可以作为参考图继续生成新图片。

### Q: 标注工具支持触摸屏吗？
A: 部分支持。推荐使用鼠标或触控笔以获得最佳体验。
