# 项目结构

```
image_workflow_studio/
├── backend/                    # Python 后端
│   ├── app.py                 # Flask API 服务
│   ├── requirements.txt       # Python 依赖
│   └── outputs/               # 生成的图片（自动创建）
│
├── frontend/                   # React 前端
│   ├── src/
│   │   ├── components/
│   │   │   ├── Canvas.jsx            # 主画布组件
│   │   │   ├── ImageNode.jsx         # 图片节点组件
│   │   │   ├── ConfigPanel.jsx       # API 配置面板
│   │   │   ├── GenerationPanel.jsx   # 生成面板
│   │   │   └── Toolbar.jsx           # 工具栏
│   │   ├── styles/
│   │   │   └── app.css               # 全局样式
│   │   ├── utils/
│   │   │   └── api.js                # API 调用封装
│   │   ├── App.jsx                   # 根组件
│   │   └── main.jsx                  # 入口文件
│   ├── index.html                    # HTML 模板
│   ├── package.json                  # 前端依赖
│   └── vite.config.js                # Vite 配置
│
├── electron/                   # Electron 配置（待添加）
│
├── 启动.bat                    # Windows 启动脚本
├── 使用说明.md                 # 使用文档
└── README.md                   # 项目说明
```

## 已完成的功能

✅ 后端 Flask API（集成 Google Gemini）
✅ 前端 React + React Flow 无限画布
✅ 图片节点组件（支持生成状态显示）
✅ 生成面板（提示词、参数、参考图）
✅ API 配置面板
✅ 完整的 CSS 样式
✅ 启动脚本
✅ Electron 桌面应用配置
✅ Electron 打包配置

## 可用功能

- Web 开发模式：使用 `启动.bat`
- Electron 开发模式：使用 `启动-Electron开发模式.bat`
- 打包成 EXE：运行 `npm run electron:build`
