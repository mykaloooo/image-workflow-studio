# 家居产品 Bento 卖点海报工作流

状态：草案，基于 Super-i 产品卖点海报方法改造成家居产品专用流程

## 1. 流程定位

本流程用于把家居产品素材、参考风格图和卖点文案，转化为可执行的 Bento Box 卖点海报方案。适用范围包括床垫、沙发、餐桌、椅子、灯具、柜子、软装单品和家居小件。

本流程学习的是 Super-i 的拆分逻辑，不复刻 3C 产品表达。家居产品的核心不是参数炫技，而是空间氛围、材质质感、尺寸比例、生活方式、工艺细节和搭配关系。

核心原则：

- 先做产品事实提取，再做视觉策划。
- 先生成无字视觉素材，文字卖点和版式后期排版。
- 不让模型直接生成复杂中文详情页。
- 每个 Bento 模块都要独立保存标题、副标题、视觉描述、比例和提示词。
- 某个模块不满意时，只替换该模块，不整张重做。
- 家居产品必须保持原始外形、颜色、材质、结构和尺寸关系。
- 人物只作为生活氛围辅助，默认避免正脸、复杂手部和夸张动作。

## 2. 和 3C 卖点海报的区别

| 维度 | 3C / 电器逻辑 | 家居产品逻辑 |
|---|---|---|
| 购买动机 | 参数、功能、技术、性能 | 好看、舒服、耐用、好搭、适合我家 |
| 主视觉 | 产品科技感展示 | 产品进入真实空间后的理想状态 |
| 模块重点 | 内部结构、功能动效、参数对比 | 空间适配、材质细节、工艺收口、尺寸比例 |
| 风格语言 | 科技光效、蓝色轮廓光、爆炸图 | 自然窗光、低饱和配色、材质微距、室内杂志风 |
| 风险 | 文字乱码、参数错 | 产品变形、比例错、材质跑偏、空间不匹配 |

## 3. 标准输入

每次启动本流程前，尽量收集以下素材：

| 输入 | 说明 |
|---|---|
| 产品主图 | 锁定整体外观、颜色、比例和角度 |
| 产品细节图 | 锁定材质、纹理、包边、缝线、五金、支脚等 |
| 产品结构图 | 锁定内部层次、支撑结构、组合方式 |
| 尺寸信息 | 长宽高、厚度、坐深、离地高度、适配空间 |
| 材质信息 | 木材、布艺、皮革、黄麻、金属、石材、玻璃等 |
| 空间参考图 | 目标家居风格、光线、配色、软装氛围 |
| 卖点文案 | 商品详情页、供应商说明、口播文案、旧海报文案 |
| 目标人群 | 儿童、青少年、年轻家庭、老人、高端客户、酒店民宿等 |
| 目标用途 | 详情页、主图、卖点图、小红书种草图、招商图 |

如果素材不足，先补产品事实，不直接进入海报策划。

## 4. 工作顺序

1. 产品事实提取
2. 参考风格逆向
3. 家居购买决策维度拆解
4. Hero 主视觉策划
5. Bento 模块规划
6. 分模块生图提示词生成
7. 整张草图或分模块试生成
8. 删除文字，得到无字底图
9. 后置文字排版
10. 单模块重生和替换
11. 质检复盘
12. 导出最终图或导入 Eagle

## 5. Agent 1：家居视觉策划总监

### 5.1 角色设定

你是一位资深家居电商视觉策划总监，同时具备空间软装搭配、家居产品摄影、详情页转化设计和生活方式内容策划经验。你擅长把产品图、材质图、尺寸信息和卖点文案，转化为适合 AI 生图和后期排版的《家居产品高转化视觉执行方案》。你关注的不只是功能，而是产品放进真实家庭空间后的美感、舒适度、信任感和购买理由。

### 5.2 输入

- 参考图
- 产品主图
- 产品细节图
- 产品结构图
- 尺寸和材质信息
- 卖点文案
- 目标人群
- 目标风格
- 输出用途

### 5.3 输出结构

```text
第一阶段：参考风格逆向
第二阶段：产品气质定位
第三阶段：购买决策卖点分层
第四阶段：Hero 主视觉策划
第五阶段：Bento 模块策划
第六阶段：设计建议
第七阶段：生图风险提示
```

### 5.4 参考风格逆向

必须分析：

- 色彩：背景色、主体色、材质色、强调色、文字色。
- 光影：自然窗光、柔和侧光、逆光、阴影强弱、空间通透感。
- 空间：墙面、地面、窗帘、软装、留白、尺度感。
- 排版：Hero 占比、网格数量、卡片圆角、间距、信息密度。

### 5.5 产品气质定位

输出产品在视觉上的关键词，例如：

- 清爽儿童成长睡眠
- 高端酒店感
- 奶油风温柔家居
- 原木自然风
- 轻法式精致感
- 侘寂低饱和
- 小户型实用收纳
- 宠物友好耐脏

### 5.6 购买决策卖点分层

家居产品优先从以下维度拆卖点：

| 维度 | 说明 |
|---|---|
| 空间适配 | 放进什么房间，适合多大空间，会不会压迫 |
| 材质质感 | 纹理、触感、光泽、厚薄、亲肤感 |
| 工艺细节 | 缝线、收口、边角、五金、支脚、拼接 |
| 尺寸比例 | 高度、厚度、坐深、床垫厚度、通道空间 |
| 舒适体验 | 坐感、睡感、支撑、透气、回弹、包裹 |
| 生活方式 | 亲子、阅读、会客、围坐、独处、睡眠 |
| 清洁维护 | 防污、易擦、耐磨、宠物友好、可拆洗 |
| 风格搭配 | 和地板、墙面、窗帘、灯具、软装的搭配 |
| 信任背书 | 环保、检测、承重、材料、售后、安装 |

每个模块标题不超过 6 字，副标题不超过 12 字。

## 6. 家居 Bento 模块池

### 6.1 Hero 主视觉区

目标：回答“它放进我家会不会好看”。

常见画面：

- 产品位于真实家居空间中心。
- 自然窗光进入空间。
- 背景干净但有生活温度。
- 留出后期排版空间。
- 不堆杂物，不让道具抢主体。

### 6.2 空间适配模块

标题示例：

- 小户型友好
- 大宅显高级
- 客厅不压层高
- 卧室刚刚好

视觉方向：产品放在对应空间里，展示空间留白、通道和尺度关系。

### 6.3 材质细节模块

标题示例：

- 细腻木纹
- 亲肤布艺
- 温润石纹
- 清爽黄麻

视觉方向：微距材质特写，展示纹理、触感、层次和光泽。

### 6.4 工艺收口模块

标题示例：

- 圆润边角
- 精密缝线
- 稳固支脚
- 细致包边

视觉方向：边角、缝线、五金、支脚、连接处的局部特写。

### 6.5 尺寸比例模块

标题示例：

- 尺度刚好
- 坐深舒适
- 黄金高度
- 厚度扎实

视觉方向：用空间、人体局部或家具组合做比例参照，但不让人物抢画面。

### 6.6 舒适体验模块

标题示例：

- 软硬适中
- 久坐不累
- 舒适承托
- 透气不闷

视觉方向：手按、坐感、回弹、蓬松、贴合、支撑等真实体验的局部画面。

### 6.7 生活方式模块

标题示例：

- 亲子陪伴
- 围坐用餐
- 午后阅读
- 安静入眠

视觉方向：温和生活片段，可用背影、局部手部、虚化人物或无人物氛围替代。

### 6.8 清洁维护模块

标题示例：

- 好打理
- 防污易擦
- 耐刮耐磨
- 宠物友好

视觉方向：水珠滚落、轻擦表面、宠物毛发易清理、耐磨材质近景。

### 6.9 风格搭配模块

标题示例：

- 百搭奶油风
- 适配原木风
- 轻法式氛围
- 侘寂感空间

视觉方向：产品与地板、墙面、窗帘、灯具、挂画、地毯同框。

### 6.10 信任背书模块

标题示例：

- 环保用材
- 稳固承重
- 严选五金
- 尺寸清晰

视觉方向：材料剖面、结构稳定、实验室感道具、干净参数留白。具体检测结果必须以后期真实文字排版为准。

## 7. 推荐排布

| 场景 | 排布 | 说明 |
|---|---|---|
| 单品详情页卖点海报 | 1 + 2 + 3 + 3 | Hero + 两个核心大卖点 + 六个小卖点 |
| 高端家居氛围海报 | 1 + 2 + 2 | 少而高级，适合沙发、床、餐桌、灯具 |
| 材质工艺型产品 | 1 + 1 + 4 | 一个大空间图，一个大材质/结构图，四个细节格 |
| 小红书种草图 | 1 + 4 | 主场景大图 + 四个高频购买理由 |

## 8. Agent 2：家居生图 Prompt 工程师

### 8.1 角色设定

你是一位家居产品 AI 绘图提示词工程师，擅长家居空间摄影、材质微距、软装搭配、空间比例和真实产品图生成。你的任务是把《家居产品高转化视觉执行方案》里的 Hero 和每个 Bento 模块，转化成可直接用于生图的中文提示词。

### 8.2 每个模块输出格式

```text
模块名称：
构图及比例建议：
中文生图提示词：
```

### 8.3 提示词必须包含

- 主体描述
- 环境与道具
- 光影与氛围
- 摄影或渲染参数
- 产品一致性约束
- 防文字干扰指令

### 8.4 家居提示词关键词

优先使用：

- 自然窗光
- 柔和侧光
- 低饱和配色
- 米白墙面
- 原木地板
- 亚麻窗帘
- 室内设计杂志风
- 真实家居摄影
- 材质微距
- 柔和阴影
- 空间留白
- 生活感但不杂乱
- 50mm 定焦镜头
- 微距摄影
- 浅景深
- 真实照片级
- 高分辨率

谨慎使用：

- 科技感蓝色轮廓光
- 爆炸剖面
- 夸张气流线
- 复杂人物动作
- 过度未来感渲染

### 8.5 固定禁止项

每个提示词末尾默认加入：

```text
保持产品原始外形，保持产品真实颜色，保持产品真实材质，保持产品真实结构和比例，不要额外生成不存在的装饰件，不要人物正脸，不要复杂手部，不要任何文字，不要字母，不要排版，不要水印
```

## 9. 生成策略

### 9.1 草图模式

用于快速看整体方向：

```text
产品图 + 参考图 + 视觉执行方案
→ 生成整张 Bento 海报草图
```

草图只看版式和视觉方向，不直接商用。

### 9.2 无字底图模式

正式生产时优先使用：

```text
模块提示词
→ 生成无字 Hero 图和无字模块图
→ 后期排版标题、副标题、参数和图标
```

### 9.3 去字模式

如果模型生成的海报含文字污染，用简单指令得到干净底图：

```text
删除图片中的文本，保留图片主体、背景、光影、产品和网格卡片结构
```

### 9.4 单模块替换

当某个模块不满意时：

1. 找到模块 ID。
2. 读取该模块标题、副标题、视觉描述和提示词。
3. 单独生成对应比例图片。
4. 替换回 Bento 网格。
5. 保持同一套色彩、光影和卡片圆角。

## 10. 质检标准

每轮生成后检查：

| 检查项 | 标准 |
|---|---|
| 产品一致性 | 外形、颜色、材质、结构不能被改 |
| 空间合理性 | 产品比例、地面接触、透视关系合理 |
| 材质真实性 | 木纹、布艺、皮革、黄麻、金属等不能乱画 |
| 卖点匹配 | 画面能表达对应模块，不只好看 |
| 排版可用性 | 有留白，文字后置能放进去 |
| 无文字污染 | 没有乱码字、假 logo、假参数、假图标 |
| 人物风险 | 无正脸、无怪手、无抢主体的人物 |
| 商业感 | 干净、高级、可信，适合电商详情页 |
