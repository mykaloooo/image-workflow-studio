## 待实现功能

### 1. 3D 角度精确控制 ⭐⭐⭐

**需求描述**:
- 用户希望能精确控制生成图片的视角角度
- 类似专业AI设计工具的角度调整控件
- 可以调整旋转、倾斜、缩放等参数

**技术方案**:

#### 方案A: ControlNet + Stable Diffusion（精确控制）
- 使用深度图提取 + 3D变换
- 部署 Stable Diffusion 服务
- 集成 ControlNet 插件
- **优点**：精确控制角度
- **缺点**：需要额外部署，复杂度高

**工作流程**：
```
原图 → 深度估计模型 → 深度图 → 3D变换 → 新深度图 → ControlNet → 新视角图片
```

#### 方案B: Gemini API + 角度描述（近似控制）
- 前端3D预览控件
- 将角度参数转换为自然语言描述
- 使用 Gemini API 生成
- **优点**：简单，使用现有API
- **缺点**：角度不精确，依赖AI理解

**示例代码**：
```javascript
function angleToPrompt(rotateX, rotateY, rotateZ) {
  let viewAngle = '';

  if (rotateY < -30) viewAngle += '从左侧观看，';
  else if (rotateY > 30) viewAngle += '从右侧观看，';
  else viewAngle += '正面视角，';

  if (rotateX > 30) viewAngle += '俯视角度';
  else if (rotateX < -30) viewAngle += '仰视角度';
  else viewAngle += '平视角度';

  return viewAngle;
}
```

#### 方案C: 混合方案（推荐）
- 前端提供3D预览和角度调整
- 预设常用角度（正面、左45°、右45°、俯视等）
- 转换为文本描述传给API
- 标注"AI理解的近似角度"

**实现优先级**: 中
**预计工作量**: 2-3天
**依赖**: 需要评估 Gemini API 的角度理解能力
