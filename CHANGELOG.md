# 更新日志

## v2.5.0 (2026-05-03)

### 核心定位
- PC2 本地工作区确认为 `D:\2026AI\image-workflow-studio`
- Studio 继续作为图片工作流主控：负责对话、提示词沉淀、供应商配置、图片生成与本地 outputs 保存
- Eagle 插件负责图库选图、批量提交、结果导入和 `gen` 文件夹归档

### 出图通道
- 新增日本 Plus 独立出图通道 `chat2api_japan_plus`
- Plus 服务独立运行在日本服务器 `108.61.180.83:8016`
- Studio provider 已支持 `codex-gpt-image-2` / `gpt-image-2`
- Plus 池与原 Pro 主通道分离，不共用账号池和配置
- 该通道定位为卡卡专用的稳定生图/图生图渠道

### Eagle 插件集成
- `image-batch-gen` 插件已接入 Studio `/api/generate`
- 支持从 Eagle 选图后批量提交图生图任务
- 生成结果默认导入源图对应的 `gen` 文件夹
- 新增供应商管理入口，可从插件打开 Studio provider 设置页
- 插件可刷新供应商和模型列表，避免手动切配置

### Flow Web 后台链路
- Studio 已接入 `flow_web_local` provider
- Flow Web Automator 可通过后台 worker 生成图片并回传到 Studio outputs
- Eagle 插件 Flow 参数已按真实网页菜单收敛：比例、张数、1K 回传
- Flow 候选图已写入元数据，并支持后续 2K / 4K 二次下载链路

### 稳定性修复
- OpenAI-compatible 文生图和图生图请求超时从 180 秒提升到默认 600 秒
- 新增环境变量 `STUDIO_OPENAI_IMAGE_TIMEOUT_SECONDS`
- 修复大尺寸 high 图生图在远端约 180 秒成功、本地刚好超时导致 Eagle 拿不到结果的问题
- Studio 后端已增加图像请求 timeout 日志，方便按开始时间排障

### 账号与额度可视化
- 日本 Plus 通道补充最小管理页
- 可查看账号总数、正常 Plus 数、总剩余额度和最后刷新时间
- 管理页不内置服务 Key，不在文档或日志中保存 token/key/password 明文

### 后续计划
- 新增“展会产品提取图顾问” agent
- 目标工作流：展会照片分析 -> 多轮对话确认产品提取方案 -> 生成产品提取图提示词 -> 交给图库/Plus 出图池生成
- 视觉理解和图片打标建议后续单独接 vision provider；Plus 通道优先用于出图，不承担主视觉分析

## v2.4.0 (2026-04-29)

### Eagle 批量生图恢复链路
- Studio 后端新增生成恢复账本 `generation_recovery.json`
- Eagle 插件提交任务时带 `task_meta`，导入成功后回传 `recovery_id`
- 支持巡逻补拉和 Eagle 入库确认，降低远端成功但本地未入库的丢图风险
- 远端图片清理增加保护：仅清理已确认 `eagle_imported` 且超过缓冲时间的图片

### 画布联动
- Eagle 插件新增“转画布”入口
- 可把当前选中的 Eagle 图片上传到 Studio，并在无限画布中自动生成图片节点
- 支持后续在 Studio 内进行多图参考、局部细节提取和二次精修

### 尺寸规则
- Studio 与 Eagle 插件统一 OpenAI 图像尺寸约束
- 长边不超过 3840，总像素控制在服务允许范围内
- 修复小尺寸低于最小像素和大尺寸四舍五入溢出问题

## v2.1.0 (2026-02-04)

### 🎨 新功能

#### 图片标注编辑器
- 新增全屏图片标注编辑功能
- 支持多种绘图工具：
  - ✏️ 画笔工具 - 自由绘制
  - ↗️ 箭头工具 - 添加指向性标注
  - ▭ 矩形工具 - 框选区域
  - Tt 文字工具 - 添加文字说明
- 支持撤销/重做操作
- 可调整颜色、线条粗细、字体大小
- 标注后的图片可继续用于工作流

### 🔧 技术改进
- 集成 fabric.js 绘图库
- 新增 `ImageAnnotationEditor.jsx` 组件
- 在 `ImageNode` 中添加标注编辑入口
- 在 `Canvas` 中添加 `handleUpdateImage` 回调
- 优化图片更新流程

### 📝 文档
- 新增 `IMAGE_ANNOTATION_FEATURE.md` 功能说明文档
- 更新 `CHANGELOG.md`

### 🎯 使用方式
1. 点击图片节点打开详情面板
2. 点击 "✏️ 标注" 按钮
3. 使用工具进行标注
4. 保存后图片自动更新
